### compass
The 'compass' task calls 'compass compile' at compass project root location (config.rb home).

##### In bundle.wrangler.config.*:
```
tasks:
  compass:
  	# Compass project root dir
    configrb: null # config.rb home
```

##### In {bundle}.*:
```
  compass:
  	# Compass project root dir
    configrb: null # config.rb home
```

##### Options:
None.

##### Flags:
None.
